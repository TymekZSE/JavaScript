1) Wymień cztery sposoby wyprowadzania danych w JS

document.write() - wypisuje treść bezpośrednio w dokumencie HTML.

alert() - wyświetla komunikat w oknie alertu.

Okienka formularza - np. używając value dla input.

Tabela - wstawianie danych do tabeli HTML.

2) Czym jest funkcja w JS?

1. Funkcja to zbiór instrukcji, które wykonują określone operacje.

2. Funkcja wykonuje kod po jej wywołaniu poprzez podanie jej nazwy.

3. Funkcja może przyjmować argumenty, np. function suma(a, b).

4. Może zwracać wartość używając słowa kluczowego return.

3) Omów metody getElementsByName, getElementById, getElementsByTagName

Metoda Opis
getElementById("id") Dostęp do elementu o podanym ID.
getElementsByTagName("znacznik") Dostęp do wszystkich elementów o danym znaczniku np. p, div.
getElementsByName("nazwa") Dostęp do elementów o podanej nazwie (name).

4) Omów właściwości innerHTML, innerText, className

innerHTML - pozwala na odczyt, zmianę oraz usunięcie kodu HTML wewnątrz elementu.

innerText - określa tekst, który zostanie umieszczony w elemencie, ignorując HTML.

className - pozwala przypisać klasę CSS do elementu.
